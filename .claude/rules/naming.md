# Naming Convention

Rationale: `.claude/rationale/naming.md`

"Ze" = "The" with a French accent. Use "ze" where "the" works grammatically.

| Context | Use |
|---------|-----|
| CLI binary | `ze` |
| BGP config YANG | `ze-bgp-conf` |
| BGP JSON format | `ze-bgp` |
| Go variables | `ZeBGPConf*` |
| YANG: config `-conf`, API `-api` |
