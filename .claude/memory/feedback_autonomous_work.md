Do not ask questions or wait for user confirmation when writing code. Complete the work autonomously.

**Why:** User delegates hard work and won't be available to answer questions. Asking blocks progress.

**How to apply:** When implementing specs or fixing issues, make decisions and keep going. Only stop when truly blocked with no viable path forward. Present completed work, not intermediate checkpoints.
