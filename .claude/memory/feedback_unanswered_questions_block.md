When I ask a question and the user's next message doesn't answer it (they say "go ahead", "let's do that", or move to a different topic), I must not silently assume an answer and proceed.

**Why:** User pointed out (2026-03-22) that missed questions lead to misaligned work. I proceed on assumptions, the user hasn't agreed, and the result doesn't match what they wanted. The wasted effort is worse than the brief pause to re-ask.

**How to apply:**
- Before starting work after an unanswered question: "Before I proceed -- I asked [X] earlier and I want to make sure we're aligned. [Restate question briefly]."
- If the question was minor and has an obvious default, state the assumption explicitly: "I'm assuming [Y] since we didn't settle this -- flag it if that's wrong."
- Never silently pick an answer to an open question.
